# mswd
一、解决什么需要
	阶段性不定期存储不同结构的数据，且不同批次的数据之间可能存在关联性。查询时可通过单个字段值查询到所有关联数据。可通过指定字段值导出数据行形成字典。可通过指定批次名称导出数据。
